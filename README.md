# Gravitational-Waves
Extracted Publicly available data using python plugins such as astropy, gwpy and gwosc from LIGO to Plot the strain curve, merger graph and the spectrogram of the a compare different Gravitational wave events like GW250114 (binary black hole merger) and GW170817 (binary neutron star merger). the code is in the GRAV_WAVES.ipynb file and the resulting plots of the 2 merger events with comparisons are in the BNS and BB merger.pdf file
